## How I log progress on GitHub

Every time a **project milestone** is done:

1. Code + tests green in that project repo  
2. Update that repo’s `MILESTONES.md`  
3. Commit: `feat(mX): …` and tag: `milestone-X`  
4. Update **this** repo’s [`PROGRESS.md`](./PROGRESS.md)  
5. (Optional) Create a GitHub Release on the project repo named `Milestone X`
